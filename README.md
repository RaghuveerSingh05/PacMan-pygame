## PAC-MAN - Classic Arcade Game

A recreation of the classic Pac-Man arcade game built with Python and Pygame. Navigate through the maze, collect dots, avoid ghosts, and use power pellets to turn the tables on your pursuers!

## Features

- **Classic Maze Layout** - Authentic 28x31 grid maze design
- **Intelligent Ghost AI** - Each ghost has unique behavior:
  - **Blinky (Red)** - Directly chases Pac-Man
  - **Pinky (Pink)** - Ambushes from ahead
  - **Inky (Cyan)** - Complex flanking behavior
  - **Clyde (Orange)** - Chases when far, scatters when close
- **Ghost Modes** - Scatter and Chase phases alternate for strategic gameplay
- **Power Pellets** - Eat special dots to turn ghosts blue and eat them for bonus points
- **Smooth Animation** - 60 FPS gameplay with smooth Pac-Man mouth movement
- **Score System** - 10 points for dots, 50 for power pellets, 200+ for ghosts
- **Lives System** - 3 lives with visual indicators
- **Progressive Difficulty** - Ghosts get faster as you level up

##  Controls

- W : Up
- A : Left
- S : Down
- D : Right
- Space : Pause 
- Enter : Restart (when Game Over!)
- Esc : Quit

## Requirements

- Python 3.12.5
- Pygame 2.6.1
